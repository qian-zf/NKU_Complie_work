## 词法分析
### 词法分析的主要任务

从左到右扫描源程序的字符，识别出各个**单词**，确定单词的**类型**。将其转换为统一的机内表示——词法单元token，最终生成token序列。

$$
token : <种别码，属性>
$$

### 词法单元token

| 单词类型 | 种别                      | 种别码                             |
| ---- | ----------------------- | ------------------------------- |
| 关键字  | if、while...             | 一词一码（因为关键字可以枚举，每一个关键字可使用单独的种别码） |
| 标识符  | 变量名、数组名...              | 多词一码，靠属性区分                      |
| 常量   | 整型、浮点型、字符型......        | 一型一码，属性标识具体数值                   |
| 运算符  | 算数运算符、关系运算符、逻辑运算符...... | 一词一码或一型一码（根据具体实现方式）             |
| 界限符  | ; () {}...              | 一词一码                            |
